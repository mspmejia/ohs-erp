# OHS Training GT — Sistema ERP

Sistema de gestión empresarial (ERP) para **OHS Training GT**, empresa guatemalteca de salud y seguridad ocupacional.

## Acceso al sistema

🔗 **[Abrir ERP](https://mspmejia.github.io/ohs-erp/OHS_ERP.html)**

> Credenciales de acceso configuradas internamente. Contactar al administrador del sistema.

## Módulos disponibles

| Módulo | Descripción |
|--------|-------------|
| Dashboard | Resumen general del negocio |
| Reportes | Análisis de ventas, compras e inventario |
| Presupuestos | Planificación financiera por período |
| Proyectos | Gestión de proyectos y tareas |
| Cotizaciones | Creación y seguimiento de cotizaciones |
| Clientes (CRM) | Base de datos de clientes |
| Inventario | Control de stock de productos |
| Finanzas | Ingresos, gastos y cuentas |
| Producción | Fabricación de productos propios |
| Compras | Órdenes y pagos a proveedores |
| Ventas | Facturación y cobros a clientes |
| Contabilidad | Libro diario y asientos contables |
| Configuración | Usuarios, unidades, empresa, respaldo |

## Tecnología

- Aplicación web de un solo archivo (Single Page Application)
- React 18 + Babel Standalone (sin dependencias de build)
- Datos almacenados en `localStorage` del navegador
- Compatible con cualquier navegador moderno (Chrome, Edge, Firefox)

## Archivos principales

| Archivo | Descripción |
|---------|-------------|
| `OHS_ERP.html` | Sistema ERP completo |
| `restaurar_cotizacion.html` | Herramienta de recuperación de cotización |
